tidc - tweak import data composer

This is an assistant tool for the Chrome extension tweak. Specifying "Target Domain" and "Select Zip" will generate JSON that can be imported by tweak based on the directory tree in the Zip file. Click the "Save JSON" button to save the JSON.

Components
  • HTML5 Boilerplate
  • unzip.min.js
  • Tailwind CSS
  • Flowbite
